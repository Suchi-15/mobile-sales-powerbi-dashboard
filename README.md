# Mobile Sales Analysis Dashboard (Power BI)

## Project Overview
This project analyzes mobile phone sales performance across multiple brands and regions using Power BI.

The dashboard provides insights into:
• Total Sales
• Brand-wise Sales Performance
• Monthly Sales Trends
• Regional Sales Distribution

## Tools Used
- Power BI
- Excel / CSV
- Data Visualization
- Data Modeling
- DAX
